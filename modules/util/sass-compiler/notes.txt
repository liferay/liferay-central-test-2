Each native library requires a specifc environment in order to build.
Once on the correct environment, run the related ant build task for creating
the library, e.g. "ant build-linux-x86-64" for building the linux 64 library.

resources/darwin/libsass.dylib
	- install OSX 10.7.5
	- login to developer.apple.com
	- download "Command Line Tools (OS X Mountain Lion) for Xcode - April 2013"
	- install package xcode462_cltools_10_76938260a.dmg
	- go to sass-compiler source
	- ant build-darwin

resources/linux-x86/libsass.so
	- install ubuntu 10.04 LTS 32bit http://old-releases.ubuntu.com/releases/10.04.3/ubuntu-10.04.4-desktop-i386.iso
	- install gcc 4.6
	- install g++ 4.6
	- go to sass-compiler source
	- ant build-linux-x86

resources/linux-x86-64/libsass.so
	- install ubuntu 10.04 LTS 64bit http://old-releases.ubuntu.com/releases/10.04.3/ubuntu-10.04.4-desktop-amd64.iso
	- install gcc 4.6
	- install g++ 4.6
	- go to sass-compiler source
	- ant build-linux-x86-64

resources/win32-x86/sass.dll
	- install Windows 7 64bit
	- download tdm-gcc 32bit toolchain
	- http://sourceforge.net/projects/tdm-gcc/files/TDM-GCC%20Installer/tdm-gcc-4.9.2.exe/download
	- click "Create"
	- install with all defaults
	- go into installation folder C:\TDM-GCC-32\bin
	- copy gcc.exe to new file named cc.exe
	- go to sass-compiler source
	- ant build-win32-x86

resources/win32-x86-64/sass.dll
	- install Windows 7 64bit
	- download tdm64-gcc 64bit toolchain
	- http://sourceforge.net/projects/tdm-gcc/files/TDM-GCC%20Installer/tdm64-gcc-4.9.2-3.exe/download
	- click "Create"
	- install with all defaults
	- go into installation folder C:\TDM64-GCC-32\bin
	- copy gcc.exe to new file named cc.exe
	- go to sass-compiler source
	- ant build-win32-x86-64
