Liferay 7.0 Community Edition

Thank you for choosing Liferay. This release further expands Liferay's ability to be used in an extremely broad range of applications for web sites, portals, and collaboration services. Remember the Liferay platform is built by and for a very large open community. Make sure to participate by contributing back to the Liferay Community.

I. New Features

Liferay 7.0 includes many new features targeting end users (power users), systems administrators and developers for enterprise deployments. Each area includes multiple features designed to support the most complex deployments. Liferay includes a web experience management suite, collaboration tools, and business productivity apps all wrapped up in a platform framework. These features combine to allow Liferay to support an extremely broad range of use cases.

II. Community Resources

Find information, participate in discussions, and contribute back to the community in the Participate section of the Liferay Developer Network (LDN) at https://dev.liferay.com. Make sure to "Like" Liferay on Facebook to collaborate with others and to find people in your field or specialty who are also using Liferay. Also make sure to "Follow" us and the community on Twitter which uses TweetRay to re-post all tweets from the community who use the tag #Liferay.

  1. Installation and Documentation
    1. To get up and running quickly for evaluation purposes (not for production), you can follow the Quick Start instructions.
    2. For comprehensive installation instructions to support enterprise production deployments, refer to the Deployment Guide.
    3. Liferay's User Guide shows you how to build sites using Liferay's features.
    4. Liferay's Learning Paths are a great place to learn how to develop on and customize Liferay, especially if you're new to Liferay development.
    5. As you gain experience developing on Liferay, you'll want to dive deep into Liferay technologies and frameworks via Liferay's Tutorials.
    6. If you create apps that you want to share or sell, visit the Developer Network's Distribute section and follow the How to Publish guide to publish the apps to the Liferay Marketplace.
  2. Liferay's Open Source Community
    1. Go to liferay.org to search for answers within the forums, to post answers for others, and learn about Liferay topics and terms in Liferaypedia.
  3. Bug Reporting
    1. While the Liferay Community cannot ensure all reported bugs will get fixed or receive a response right away, contributing bug information to Liferay supports everyone and is one of the most valuable ways you can contribute back to the Liferay Community. If you find a bug, please log as much information as possible into a new ticket at: http://issues.liferay.com.
  4. Suggesting New Features
    1. The Liferay open source project relies on its community for many of the evolutionary and revolutionary ideas that eventually find their way into a release. Liferay strongly believes in empowering the community not only to be able to create change through feature requests but also to feel responsible for socializing, recommending, and implementing their own and others' ideas. If you have a great idea for Liferay, check out the Ideas Dashboard, which provides an easy way to create, view, and recommend feature requests, and help steer the direction of Liferay!
  5. Liferay Marketplace
    1. The Liferay Marketplace is an exciting hub for sharing, browsing and downloading Liferay-compatible applications. As enterprises look for ways to build and enhance their existing platforms, developers and software vendors are searching for new avenues to reach this market. Marketplace leverages the entire Liferay ecosystem to release and share apps in a user-friendly, one-stop site. Browse the available apps, or visit the Developer Network's Distribute section to learn how to build your own apps!
  6. Staying in Touch
    1. Follow us on Twitter:
      • @Liferay tweets the Liferay's latest announcements.
      • @Liferaydocs tweets about new articles.
      • @Liferayeng tweets happenings from the core engineering team.

IV. The Liferay Development Team

This release was produced by Liferay's worldwide portal engineering team, and involved many hours of development, testing, writing documentation, and working with the wider Liferay community of customers, partners, and open source developers. We are glad you have chosen to use Liferay, and hope that it meets or exceeds your expectations!

In addition to Liferay's engineering staff, a special thanks goes to the many open source developers who volunteered their time and energy to help with the release, whether it was bug fixing, idea generation, documentation, translations, or other contribution that helped to improve this release. Check out the Community Contributor Hall of Fame and be sure to thank them if you run across them in the community!
