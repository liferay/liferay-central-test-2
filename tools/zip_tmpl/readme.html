I. New Features

Liferay Portal 6.1 EE includes many new features targeting end users (power users), systems administrators and developers for enterprise deployments. Key themes and features in the new Liferay Portal 6.1 EE include:

Anytime Access to the Enterprise

  1. New Liferay Sync: Document sharing becomes a seamless experience with online and offline synchronization to desktop and mobile devices.
  2. Third-party Integration: Documents can be accessed in one place through enterprise integration with Documentum, SharePoint, and other popular third-party CMIS repositories.

Better, Easier Content Organization

  1. Advanced Content Management: Drag-and-drop document and media libraries centralize and simplify content management.
  2. Dynamic Video and File Previews: Users can preview documents, images and videos before downloading them.

Flexible Multi-User Publishing

  1. User Customizable Sites: Administrators can now designate certain pages and sections of a page to be personalized by users.
  2. Multiple Website Versions: Multiple site versions can be edited simultaneously and scheduled for publication.
  3. Affiliated Sites Management: Liferay pages and websites maintain links to their original site templates, making it easier to propagate changes to associated sites and pages.

Enhanced Productivity

  1. New Workflow and Forms Designers: Business users can use these powerful visual editors to create web forms and workflows interactively in the browser.
  2. User Data Lists: Users can create custom lists online that can be tied to form submissions to model business processes such as inventory management, vacation requests, or sales order submissions.
  3. Advanced Workflow: Workflow processes can be applied to documents to enhance and automate approval and editing processes.

Additional documentation can be found on the Liferay 6.1 User Guide.

  3. Liferay Enterprise Support Application (LESA)

    1. LESA is Liferay's web based multifaceted custom support issue tracker. LESA allows a quicker, visible escalation process while boasting an exceptional user interface that makes it easy to use. Its ability to issue multiple assignees also means more people to work on a ticket at one time.

      In order to access LESA, make sure you have a valid Liferay.com account with access to the Liferay Enterprise Support Application and Customer Portal.

      If you do not have a Liferay.com account, please fill out the contact form. Our Liferay Support Administrator will contact you with your username and password; this account will be setup with access to LESA and the Customer Portal. If you encounter any issues accessing either LESA or the Customer portal, please email provisioning@liferay.com.
